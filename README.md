# SoloStandup

A browser-based utility designed to solve "blank page syndrome" before 
daily standup meetings. SoloStandup provides a structured, distraction-free 
environment to draft your "Yesterday, Today, and Blockers" updates.

## Features

* ** distraction-Free UI:** A clean, flat design using a curated Slate color 
    palette.
* **Persistent Drafting:** Uses LocalStorage to automatically save your 
    inputs, so you never lose your draft if the tab closes.
* **One-Click Formatting:** Instantly copies your update to the clipboard 
    pre-formatted with emojis and bolding suitable for Slack, Discord, or 
    Teams.
* **Zero Dependencies:** Built with vanilla HTML, CSS, and JS. No build 
    steps required.
* **Mobile Responsive:** Fully functional on mobile devices for on-the-go 
    updates.

## How to Run

1.  Download the `index.html` file.
2.  Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  Start typing your update.
